# 行业分析参考框架

## 一、主要赛道分析模板

### 1. 互联网/AI 赛道

**细分方向：**
- AI 工具开发与变现（SaaS、API、插件）
- AI 内容创作（文案、图片、视频生成服务）
- AI 培训与咨询（帮助企业/个人应用 AI）
- AI 代理（Agent）开发与部署
- 数据标注与模型微调服务

**关键判断指标：**
- 底层模型能力是否足够支撑商业应用
- 目标客户是否有真实付费意愿
- 是否有持续的技术壁垒还是很容易被复制
- 获客成本与客户终身价值的比值

### 2. 内容创作/自媒体赛道

**细分方向：**
- 短视频（抖音、快手、视频号）
- 图文（小红书、公众号、知乎）
- 播客/音频
- 直播带货
- 知识付费

**关键判断指标：**
- 平台当前的流量分配机制（是否对新人友好）
- 内容类型的竞争饱和度
- 变现路径是否清晰（广告、带货、知识付费、品牌合作）
- 内容资产是否可以长期复利

### 3. 电商/跨境赛道

**细分方向：**
- 国内电商（淘宝、拼多多、抖音电商）
- 跨境电商（亚马逊、TikTok Shop、Temu、SHEIN）
- 独立站
- 社交电商/社群团购
- 二手/闲置交易

**关键判断指标：**
- 选品能力与供应链控制
- 平台政策稳定性
- 物流与售后成本
- 市场差异化空间

### 4. 自由职业/技能变现赛道

**细分方向：**
- 编程开发（外包、产品开发、技术咨询）
- 设计（UI/UX、品牌设计、平面设计）
- 写作翻译（文案、翻译、Technical Writing）
- 营销咨询（SEO、SEM、社媒运营）
- 专业服务（法律、财务、心理咨询）

**关键判断指标：**
- 技能稀缺性与市场需求匹配度
- 单价天花板与时间利用率
- 能否从"卖时间"升级为"卖产品"
- 是否可以建立个人品牌提升溢价

### 5. 实体/本地服务赛道

**细分方向：**
- 餐饮（小吃、预制菜、外卖）
- 本地生活服务（家政、维修、美业）
- 教育培训（线下 + 线上结合）
- 新零售（社区团购、自动售卖）

**关键判断指标：**
- 本地市场需求与竞争密度
- 标准化程度与可复制性
- 人力成本与管理难度
- 政策与合规要求

### 6. 投资理财赛道

**细分方向：**
- 股票/基金投资
- 加密货币/Web3
- 房产投资
- 另类投资（收藏品、版权等）

**关键判断指标：**
- 风险收益比
- 流动性
- 所需知识门槛
- 宏观经济周期位置

## 二、通用分析工具箱

### PEST 宏观分析
- **P**olitical（政策）：相关政策法规、监管态度
- **E**conomic（经济）：宏观经济环境、消费趋势
- **S**ocial（社会）：人口结构变化、消费观念、文化趋势
- **T**echnological（技术）：技术变革、基础设施变化

### 波特五力分析（简化版）
- 竞争者多不多？→ 竞争激烈度
- 新人容不容易进？→ 进入壁垒
- 能不能被替代？→ 替代品威胁
- 客户有没有选择权？→ 买方议价能力
- 供应商会不会卡脖子？→ 供方议价能力

### PMF（产品市场匹配）快速检验
- 用户愿不愿意为此付费？（付费意愿）
- 用户会不会主动推荐给别人？（自传播性）
- 用户离开后会不会不习惯？（粘性/依赖度）

## 四、风险评估矩阵

### 风险分类

| 风险类型 | 描述 | 预防方法 |
|---------|------|---------|
| 政策风险 | 监管变化导致赛道消失 | 选择合规方向，关注政策动向 |
| 平台风险 | 平台规则变化、封号 | 不依赖单一平台，建立私域 |
| 技术风险 | 技术迭代导致方案过时 | 关注技术趋势，保持学习 |
| 市场风险 | 竞争加剧、需求萎缩 | 持续创新，建立护城河 |
| 资金风险 | 投入无法收回 | 小步验证，控制单次投入 |
| 时间风险 | 投入大量时间但无回报 | 设定止损期限，定期复盘 |
| 认知风险 | 被错误信息误导 | 多源验证，独立思考 |
| 骗局风险 | 遇到诈骗或割韭菜 | 识别骗局特征（见下方） |

### 骗局/割韭菜识别清单

以下特征出现 3 个以上的，大概率是骗局或割韭菜：

- ☑ 承诺高收益、保证赚钱
- ☑ 需要先交高额加盟费/培训费
- ☑ 强调"躺赚""零门槛""月入10万"
- ☑ 收入主要靠拉人头/发展下线
- ☑ 教你做某事的人比做这事赚钱的人多
- ☑ 不透明的商业模式，说不清楚钱从哪来
- ☑ 制造紧迫感"名额有限""马上涨价"
- ☑ 展示大量收入截图但无法验证
- ☑ 社群里全是"感恩""暴富"的话术
