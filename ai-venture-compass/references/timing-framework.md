# 时机窗口判断框架

## 核心原则

> AI 机会最大的时刻：技术刚刚"足够好"，但大多数人还不知道怎么用。

判断一个 AI 机会的时机，本质上是判断它在**技术成熟度 × 市场认知度**坐标系中的位置。

---

## 框架一：AI 机会 S 曲线

```
市场渗透率
    │                              ████ 成熟/红海
    │                         ████
    │                    ████        ← 你应该在这里进入
    │               ████
    │          ████                  ← 太早（市场不成熟）
    │     ████
    │████
    └─────────────────────────────→ 时间
      探索期  萌芽期  黄金期  成长期  成熟期  红海期
```

### 各阶段特征

| 阶段 | 特征 | 进入建议 |
|------|------|---------|
| 探索期 | 技术不稳定，几乎无市场 | 观察，不投入 |
| 萌芽期 | 技术可用，少数人在玩 | 小额试水，学习 |
| **黄金期** | **技术成熟，主流市场刚开始接受** | **全力进入** |
| 成长期 | 竞争增加，但市场快速扩大 | 有差异化可进 |
| 成熟期 | 充分竞争，价格战 | 谨慎，需强差异化 |
| 红海期 | 巨头主导，利润极薄 | 避开或转型 |

## 框架二：五项时机信号

### 信号1：技术可靠性（☑ 可用了吗？）
- 工具是否已经足够稳定，能交付给客户？
- 有没有早期用户在真实使用并付费？
- Bug 和局限是可接受的还是致命的？

**判断**: 如果核心功能 80% 时候能正常工作 → 够用

---

### 信号2：竞争密度（☑ 还有位置吗？）
搜索关键词：`[方向] site:producthunt.com` 或 `[方向] 创业`

| 竞争状况 | 含义 |
|---------|------|
| 几乎搜不到相关产品 | 太早，或无市场 |
| 能找到3-10个玩家 | 黄金期信号 |
| 大量产品，但都很通用 | 有垂直细分机会 |
| 头部产品融资亿元+ | 赛道成熟，需要强差异化 |
| 巨头已入场（微信/阿里） | 注意，可能是机会也可能是死亡信号 |

---

### 信号3：付费验证（☑ 有人愿意付钱吗？）
这是最重要的信号。

- 在 Reddit/X/小红书搜索用户在为类似问题付多少钱
- 是否有人在 Gumroad/爱发电等平台为相关产品付费？
- 有没有企业客户在询价？

**判断**: 能找到 5 个愿意付费的真实用户 → 机会存在

---

### 信号4：搜索热度趋势（☑ 需求在增长吗？）
使用工具：Google Trends、百度指数、微信指数

- 过去12个月搜索量趋势：上升/平稳/下降？
- 相关关键词是否在新增？
- 媒体报道频率是否在增加？

**判断**: 12个月内搜索量增长 > 50% → 趋势确认

---

### 信号5：先发优势价值（☑ 早入场值得吗？）

并非所有领域都需要先发优势。判断标准：

**先发优势重要的场景**:
- 网络效应强的产品（用户越多越好用）
- 需要积累用户数据/反馈才能改进的产品
- SEO/内容积累型业务
- 需要建立品牌信任的服务

**先发优势不重要的场景**:
- 纯技能服务（客户看质量不看谁先做）
- 工具集成类（快速跟进即可）
- 本地化业务（地域壁垒 > 时间壁垒）

## 框架三：窗口期估算

不同机会的窗口期差异很大：

| 机会类型 | 典型窗口期 | 窗口关闭信号 |
|---------|----------|------------|
| 蹭热点产品 | 1-4周 | 热搜消退 |
| AI 工具教程 | 3-6个月/轮 | 同类教程泛滥 |
| 垂直行业 AI 工具 | 1-3年 | 大厂发布同类产品 |
| AI 增强专业服务 | 3-5年 | AI 直接取代该专业 |
| AI Agent 平台 | 2-4年 | 标准化产品出现 |
| 企业 AI 改造服务 | 5年+ | 行业普及完成 |

## 框架四：进入时机决策树

```
这个方向我感兴趣
        ↓
技术是否足够成熟可用？
    否 → 等待，6个月后再看
    是 ↓
是否有人在为此付费（哪怕少量）？
    否 → 调研需求，或换方向
    是 ↓
竞争密度如何？
    已经是红海 → 找细分或放弃
    中等竞争 ↓
我有差异化优势吗？（行业经验/渠道/特殊资源）
    没有 → 考虑换方向或先建立优势
    有 ↓
3个月内能验证（找到首个付费客户）？
    否 → 简化方案
    是 → 进入，快速测试
```

## 常见时机误判

### 误判一："太早了"的错觉
**症状**: "现在市场还没准备好"
**真相**: 大多数人高估了"市场准备好"的难度。ChatGPT 一年内从0到1亿用户——市场比你想象的更快接受新事物。
**对策**: 找10个早期用户测试，数据说话。

### 误判二："太晚了"的恐慌
**症状**: "已经有人做了，我还有机会吗？"
**真相**: 除非已经有巨头主导且你没有差异化，否则几乎不存在"太晚"。每个垂直赛道、每个地区、每个用户群都还有机会。
**对策**: 找到更细分的切入点，而不是放弃。

### 误判三：等"完美时机"
**症状**: 一直在分析，迟迟不行动
**真相**: 时机不等人，AI 行业6个月就是一个时代
**对策**: 设定"够用就行动"的阈值，先以小成本测试

## 快速时机评分（1-5分）

对某个方向，各打1-5分，总分>15才建议全力投入：

| 维度 | 评分标准 |
|------|---------|
| 技术成熟度 | 1=实验室 3=可用但不稳定 5=生产就绪 |
| 付费验证 | 1=没人付钱 3=少量付费用户 5=明确的付费需求 |
| 市场增速 | 1=停滞 3=稳定增长 5=爆炸性增长 |
| 竞争空间 | 1=红海 3=中等 5=蓝海/细分未开发 |
| 个人适配 | 1=完全不适合 3=基本适合 5=非常适合 |

**总分解读**:
- 20-25分：立即行动
- 15-19分：值得深度研究和小规模测试
- 10-14分：谨慎，先验证关键假设
- <10分：换方向
