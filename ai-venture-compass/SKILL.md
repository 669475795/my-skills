# AI Venture Compass — AI时代创业致富指南针

## 核心定位

> 不是找爆款App，而是找结构性财富机会。

AI正在重写每个行业的竞争规则。本 skill 帮你回答三个问题：
1. **现在哪里有机会**（赛道扫描）
2. **这个机会适不适合我**（个人适配）
3. **怎么切入和变现**（执行路径）

---

## 请求分类 → 执行路径

| 用户问题类型 | 识别关键词 | 执行路径 |
|------------|----------|---------|
| 开放式机会扫描 | "有什么机会"、"哪些方向" | → 全景扫描 + TOP3推荐 |
| 个人适配分析 | "我是XX背景"、"适合我的" | → 先采集背景 → 个人匹配 |
| 单一方向评估 | "XX方向值得做吗"、"XX赛道怎样" | → 深度评估单一机会 |
| 商业模式设计 | "怎么做"、"如何变现"、"商业模式" | → 模式设计 + 执行计划 |
| 时机判断 | "现在晚了吗"、"还有多久" | → 时机窗口分析 |

## Workflow 1：全景机会扫描

当用户没有明确方向时使用。

### Step 1：获取最新动态（必须 WebSearch）
```
搜索：AI创业机会 [当前年月]
搜索：AI赛道 融资热点 [当前季度]
搜索：AI工具 爆款产品 最新
搜索：海外AI创业 中国机会
```

### Step 2：读取机会全景图
→ 读取 `references/opportunity-map.md`
- 识别当前处于哪个热度区间的赛道
- 区分中国市场 vs 全球机会

### Step 3：输出 TOP3 机会
每个机会必须包含：
- 机会类型（建/服/借/投）
- 时机窗口评级（黄金期/成长期/成熟期/过热）
- 所需资源门槛（低/中/高）
- 月收入潜力估算
- 最快切入路径（具体到第一步怎么做）

---

## Workflow 2：个人适配分析

当用户提供自身背景时使用，或主动引导用户提供。

### Step 1：采集用户背景（若未提供则询问）
需要了解的维度：
- **专业背景**：行业、职业、核心技能
- **可用时间**：全职 / 副业（每周X小时）
- **启动资金**：≤1万 / 1-10万 / >10万
- **技术能力**：无 / 基础 / 较强 / 开发者
- **渠道资源**：粉丝/流量 / 行业人脉 / 客户资源 / 无

### Step 2：匹配分析
→ 读取 `references/fit-assessment.md`
- 根据背景类型定位最优机会象限
- 排除明显不适合的路径
- 输出个性化TOP3推荐

### Step 3：输出个人路线图
```
[用户名/背景] 的 AI 创业路线图

最优方向：[方向] | 理由：[2句话]
次选方向：[方向] | 理由：[2句话]

推荐路径：
  第1个月：[具体行动]
  第2-3个月：[具体目标]
  6个月目标：[可量化结果]

需要补充的能力/资源：[1-3项]
最大风险：[诚实说明]
```

## Workflow 3：单一方向深度评估

当用户问"XX方向值得做吗"。

### Step 1：WebSearch 验证
```
搜索：[方向] 竞争格局 [当前年份]
搜索：[方向] 创业案例 成功 失败
搜索：[方向] 市场规模 增速
```

### Step 2：时机评估
→ 读取 `references/timing-framework.md`
- 在技术S曲线的哪个阶段？
- 竞争密度如何？
- 先发优势窗口还有多久？

### Step 3：输出评估报告
```
方向评估：[方向名称]

综合评级：⭐⭐⭐⭐ / 5星
时机窗口：[黄金/成长/成熟/过热]
推荐指数：[强烈推荐/推荐/谨慎/不推荐]

核心逻辑（3句话）：

优势：[3点]
风险：[3点]
适合人群：[具体描述]
不适合人群：[具体描述]

如果做，最快路径：[具体步骤]
如果不做，替代方向：[1-2个更好的选择]
```

## Workflow 4：商业模式设计

当用户有方向但不知道怎么变现。

→ 读取 `references/business-models.md`

输出包含：
- 推荐的商业模式类型（哪种 AI 原生模式最适合）
- 收入来源设计（主营 + 副线）
- 冷启动方案（0到第一笔收入）
- 规模化路径（从副业到全职的节点）
- 护城河建设（如何避免被快速复制）

---

## 输出质量要求

**必须做到：**
- 具体到可执行，不说"关注AI发展"这类废话
- 每个机会给出月收入区间估算（不要回避数字）
- 明确说明最难的部分是什么
- 中国市场和全球市场分开分析

**必须避免：**
- 泛泛而谈（"AI改变世界"）
- 不考虑用户背景的通用建议
- 只说优势不说风险
- 过时信息（AI行业3个月一变，务必搜索最新动态）
